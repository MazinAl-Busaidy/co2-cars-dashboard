# EU New Passenger Cars CO2 Dashboard

An interactive Streamlit dashboard analysing average CO2 emissions per kilometre from new passenger cars across EU countries (2000-2024).

**Module:** 5DATA004W Data Science Project Lifecycle
**Data source:** European Environment Agency, SDG 13_31 indicator
